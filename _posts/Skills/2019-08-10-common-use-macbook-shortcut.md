### Make a Screenshot

`command + shift + 4`

### Change Keyboard Input Sources

`control + space`

### Open Spotlight

`command + space`

### Show/Hide Hidden Files

`command + shift + .`

### Type a Check Symbol

`Option + V`

### Show all Special Symbols

`Control + Cmd + Space`
